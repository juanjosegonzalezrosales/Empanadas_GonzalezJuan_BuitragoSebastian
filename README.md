# 🥟 Empanadas Doña Pepa

## Descripcion
stema de gestión de catálogo de empanadas desarrollado en **Python** para ejecutarse desde consola.  
El proyecto permite administrar empanadas mediante operaciones básicas de registro, consulta, edición y eliminación, almacenando la información en un archivo JSON.

Este proyecto fue desarrollado como ejercicio de **Git, GitHub y trabajo colaborativo**, aplicando control de versiones, manejo de ramas, Pull Requests y documentación del proyecto.

## ⚙️ Instalación y Ejecución

Sigue los siguientes pasos para instalar y ejecutar el proyecto en tu equipo.

1. Obtener el proyecto

    Descarga el repositorio en tu computador y ubícate dentro de la carpeta del proyecto.

2. Verificar instalación de Python

    Asegúrate de tener Python 3 instalado en tu sistema.

    python --version

    o

    python3 --version
3. Ubicarse en la carpeta del proyecto

    Desde la terminal o consola, navega hasta la carpeta donde se encuentra el proyecto.

4. Ejecutar el programa

    Ejecuta el archivo principal del sistema:

    python main.py 

    o en algunos sistemas:

    python3 main.py
